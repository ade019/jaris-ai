## ⚠️ IF SOMETHING GOES WRONG

### Common Issue: "ModuleNotFoundError"
**Solution:** Run `python -m pip install -r requirements.txt` again

### Common Issue: "PyAudio fails to install"
**Solution:** 
- Windows: Download Visual C++ build tools from Microsoft
- macOS: Run `xcode-select --install`
- Linux: Run `sudo apt-get install portaudio19-dev`

### Common Issue: "No microphone"
**Solution:** Use text mode - `python main.py --text`

### Common Issue: "pip not found"
**Solution:** Use `python -m pip` instead of just `pip`
