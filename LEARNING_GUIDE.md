# 📚 JARVIS Learning Module - Quick Reference

## 🚀 Getting Started

**Start Learning:**
```
teach me cybersecurity
```

JARVIS will show you Lesson 1 of an 8-week defensive cybersecurity course.

---

## 📖 Available Commands in Learning Mode

| Command | What It Does |
|---------|-------------|
| `next` | Move to next lesson |
| `quiz` | Take a quiz question on current week |
| `progress` | View your learning stats & scores |
| `exit` | Leave learning mode |
| `1`, `2`, `3`, `4` | Answer quiz questions (when prompted) |

---

## 📅 8-Week Curriculum

### Week 1: Foundations of Cybersecurity
- What is Cybersecurity?
- Types of Threats
- **Quiz:** 3 questions on CIA Triad, threats, confidentiality

### Week 2: Authentication & Access Control
- Passwords & Authentication (2FA, MFA)
- Access Control Basics (RBAC, Least Privilege)
- **Quiz:** 3 questions on auth types and permissions

### Week 3: Encryption & Data Protection
- What is Encryption?
- Symmetric vs Asymmetric encryption
- **Quiz:** 2 questions on encryption methods

### Week 4: Network Security
- Firewalls & Network Defense
- IDS/IPS systems
- Network Segmentation & VPNs
- **Quiz:** 2 questions on firewalls and DMZ

### Week 5: Secure Coding Practices
- Common Vulnerabilities (OWASP Top 10)
- SQL Injection, XSS attacks
- Input validation best practices
- **Quiz:** 2 questions on coding security

### Week 6: Incident Response & Monitoring
- Security Monitoring (SIEM)
- Incident Response Steps
- Detection and analysis
- **Quiz:** 2 questions on SIEM and response

### Week 7: Cloud & Infrastructure Security
- Cloud Security (AWS/Azure/GCP)
- Identity & Access Management (IAM)
- Shared Responsibility Model
- **Quiz:** 2 questions on cloud security

### Week 8: Security Mindset & Next Steps
- Building a Security Culture
- Defense in Depth
- Certifications (Security+, CEH, CISSP)
- Legitimate platforms (TryHackMe, HackTheBox)
- **Quiz:** 2 questions on security culture

## 🎯 How to Get the Most Out

1. **Read carefully** - Each lesson has key concepts highlighted
2. **Take quizzes** - Test your understanding after each week
3. **Track progress** - Use `progress` command to see improvement
4. **Review** - Lesson content stays available; reread if confused
5. **Take breaks** - Study in multiple sessions for better retention
